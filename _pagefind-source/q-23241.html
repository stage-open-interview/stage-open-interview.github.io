Design a CGOA wrapper for a C streaming binary protocol parser that maintains a persistent Parser handle across chunked input. The C API provides: typedef struct Parser Parser; Parser* parser_create(const ParserConfig*); int parser_feed(Parser*, const uint8_t* data, size_t len); int parser_flush(Parser*, char** out_json); void parser_destroy(Parser*); The parser_flush returns a malloc-allocated JSON per complete frame. Implement: 1) a C header and minimal implementation; 2) a Go binding exposing StreamParse(chunks <-chan []byte) (<-chan string, <-chan error) with proper memory ownership; 3) a Go program that streams chunks, prints per-frame JSON, and supports cancellation?
