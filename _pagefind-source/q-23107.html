You store a multiset S of integers. Updates arrive in batches of size B; each update is either insert x or delete x (guaranteed to exist for deletion). After each batch, determine whether there exists a pair of elements with difference exactly K. Propose a data structure that (i) initializes in O(N), (ii) processes each batch in amortized O(B), (iii) answers in O(1) whether such a pair exists, (iv) uses near-linear space, (v) two optimizations, and (vi) compares to re-scanning the whole set after every batch. Provide asymptotics in N, B, K?
