Implement a data structure PrefixWeightTrie: a Trie augmented to store word weights that can only increase on insert. Requirements: - insert(word, weight): create or update the word’s weight; subsequent inserts for the same word must not decrease its weight. - queryLongestPrefix(s): return a pair (L, W) where L is the length of the longest prefix of s that matches any stored word, and W is the maximum weight among all words with that prefix; if none exist, return (0, 0). Target: insert O(len(word)), query O(len(s))?
