You are given an array A[0..n-1] and an integer k. Partition the array into exactly k non-empty contiguous blocks in order. The value of a block [l..r] is the maximum subarray sum contained within that block (i.e., max over p,q with l≤p≤q≤r of sum A[p..q]). Return the maximum possible sum of block values and the endpoints of the k blocks. Propose a DP with precomputation of max-subarray sums for all intervals, the dp[i][t], base cases, reconstruction, and discuss time/space?
