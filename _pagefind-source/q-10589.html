Given a string s and an integer k, partition s into exactly k non-empty substrings in order. Each substring earns a score equal to the number of distinct characters it contains. Return the maximum total score and the cut positions (end indices of each substring). Propose a DP formulation: dp[i][t] represents the max score for the prefix s[0..i-1] split into t substrings; the transition is dp[i][t] = max_{t-1 <= j < i} dp[j][t-1] + distinctCount(j, i-1). Precompute distinctCount for all substrings, specify base cases, and discuss time/space; include reconstruction plan?
