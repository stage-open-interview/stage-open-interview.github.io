You are given a DAG with n nodes and m edges. Each edge has a color from a small set {A,B,C}. From a start node s, compute for every node v the minimum number of color changes required to reach v (a color change occurs when consecutive edges have different colors; the first edge does not count as a change). Also compute the number of distinct paths achieving that minimum modulo 1e9+7. Propose an O(m) DP using topological order and explain how to track the color state and counts?
