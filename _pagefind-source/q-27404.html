Given an array cost of length n with non-negative integers and a budget B, you start at a virtual before-first-step and may jump to i+1 or i+2, paying cost at the landing step. Count the number of distinct paths that end at index n-1 with total cost exactly B, modulo 1e9+7. Implement bottom-up DP with O(n*B) time and O(B) space and a memoized DFS variant; discuss edge cases?
