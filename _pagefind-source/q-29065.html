You're given an undirected graph with N nodes and M edges, initially arbitrary but with no multi-edges. Updates arrive in batches of size B; each update is either add edge (u,v) or delete edge (u,v) (deletions guaranteed to exist). After each batch, determine whether the graph contains at least one cycle. Propose a data structure and an amortized analysis that (i) preprocesses in O(N+M), (ii) handles each batch in O(B log N) time, (iii) answers in O(1) whether a cycle exists, (iv) uses near-linear space, (v) two optimizations, and (vi) compare to recomputing cycles from scratch after every batch. Provide asymptotics in N, M, B?
