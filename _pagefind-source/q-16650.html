Grid percolation with batched edge additions. An n×n grid, edges activate in online batches of size B (no deletions). After each batch, determine if a path exists from the left boundary to the right boundary using only activated edges. Propose a practical incremental connectivity structure that (i) processes each edge insertion near-constantly, (ii) answers in O(1) per batch, (iii) uses near-linear space, (iv) includes two optimizations, and (v) compares to rebuilding from scratch. Provide asymptotics in n,B?
