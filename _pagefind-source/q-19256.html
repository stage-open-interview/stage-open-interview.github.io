You're given a DAG with N nodes and M directed edges; each node i has a value val[i]. You must select exactly k vertex-disjoint directed paths, where a path is any directed sequence of edges, and paths may start and end at any nodes as long as they do not share vertices. Maximize the sum of values of all nodes on the chosen paths. Provide a DP formulation with topological order, define states and transitions to enforce disjointness and the exact k-path constraint, and explain reconstruction?
