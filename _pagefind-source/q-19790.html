Edge insertions in DAGs with online batches: G=(V,E) starts with no edges. Edges arrive in batches of size B; each batch preserves acyclicity (no cycles). After each batch, determine the length of the longest path (in edges) from any source to any sink. Propose a practical incremental data structure that (i) preprocesses in O(n+m), (ii) updates amortized to O(B log n), (iii) answers in O(1), (iv) uses O(n+m) space, (v) two optimizations, and (vi) a comparison to rebuilding a DP on a topological order after every batch. Provide asymptotics in n, m, B?
