Given a directed graph of boolean flags with two constraint types: (i) A -> B meaning A true forces B true, and (ii) XOR clauses among disjoint pairs (e.g., exactly one of X and Y is true), plus a required set T that must be true. Decide if there exists an assignment satisfying all constraints and produce the smallest model (fewest true flags) containing T. Provide a small 6-flag example using both constraint types and describe a practical solver that combines forward-propagation with a parity solver (2-SAT) for the XORs?
