Design a dynamic 3D weighted point set that supports insertPoint(x,y,z,w), erasePoint(x,y,z,w) (duplicates allowed), movePoint(oldX,oldY,oldZ,newX,newY,newZ) to relocate one occurrence, rangeSum3D(x1,x2,y1,y2,z1,z2) for the total weight in the axis-aligned 3D box, and kthLargestIn3DBox(x1,x2,y1,y2,z1,z2,k) for the k-th largest weight inside that box. Target average O(log^3 N) per operation; discuss coordinate compression, memory trade-offs, and handling duplicates and moves?
