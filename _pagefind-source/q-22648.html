You manage N tasks with dynamic priorities. In each batch of B updates, you may insert a new task with priority p or adjust an existing task's priority by delta. After every batch, return the IDs of the top-k tasks by priority. Propose a practical data structure that (i) preprocesses in O(N), (ii) updates amortized O(B log S) with S ≈ sqrt(N), (iii) answers top-k in O(k log (N/S)), (iv) uses near-linear space, (v) two optimizations, (vi) compares to re-sorting after every batch. Provide asymptotics in N, B, k?
