In a DAG G=(V,E) with non-negative weights, there is a source s, a sink t, and a landmark set L of size k (k ≤ 15). Find the minimum-weight s-t path that visits every landmark in L at least once. Output the total weight and the full path. Propose an algorithm leveraging a topological order that runs in O((n+m)(k+1)) time and describe how you reconstruct the path?
