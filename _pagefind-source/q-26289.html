You're given an m x n grid of integers grid[i][j]. You start at (0,0) and can move only right or down to (m-1,n-1). You have up to k vouchers that can be applied to any visited cell; applying a voucher on a cell doubles its value for that visit. How would you compute the maximum path sum and the path with voucher placements, using a DP formulation with dp[i][j][t], and how would you reconstruct the path and voucher usage?
