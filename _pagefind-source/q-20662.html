Given a rooted tree with N nodes, each node i has value w[i]. Choose exactly K pairwise disjoint connected subtrees to maximize the sum of included node values. Subtrees are connected node sets; no two overlap. Return the maximum sum and a reconstruction of the chosen subtrees (node lists). Provide a DP formulation using dp[u][t][s], where s in {0,1} indicates whether the component containing u sticks to the parent, plus transitions and reconstruction pointers?
