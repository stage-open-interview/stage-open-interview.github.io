You are given an array A of length n and an integer k. You must partition A into exactly k non-empty contiguous subarrays. For each subarray [l..r], score = number of distinct values present in A[l..r]. Return the maximum total score and the k-1 partition indices. Propose a DP: dp[i][t] = max over p in [t-1..i-1] of dp[p][t-1] + distinctCount(p+1, i). Include precomputation of distinct counts, base cases, and reconstruction. Explain time/space?
