We model a two-value CSP with 4 variables X1..X4 in {T, F, U}. Define forward-propagation rules: - R1: if X1 = T and X2 = T then X3 := T - R2: if X4 = F then X1 := F - R3: if X3 = F then X2 := F or X4 := F - R4: if X1 = U or X2 = U then X4 := U Is there an initial assignment with at least one U such that iterated application of the rules reaches a fixed point where at least one U remains and no rule is applicable? Provide one such initial state, the fixed point, and the propagation trace?
