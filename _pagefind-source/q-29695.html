Maintain N closed intervals [L_i, R_i] on a known integer universe with O(N) endpoints. Updates arrive in batches of B; each update is either add an interval [L, R] with a new ID or remove an existing interval by ID. After each batch, determine the maximum number of overlapping intervals at any point. Propose a data structure that (i) preprocesses in O(N log N), (ii) handles each batch in amortized O(B log N), (iii) answers in O(1) the max overlap, (iv) uses near-linear space, (v) two optimizations, and (vi) compares to reconstructing overlaps from scratch after every batch. Provide asymptotics in N, B?
