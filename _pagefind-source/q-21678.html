Epistemic-temporal puzzle: three agents A,B,C reason about a private proposition P. Initially P is true only in A's local worlds. Across four public announcements, exactly one agent states a truth about P, updating knowledge for all. Log: t0 A: P, t1 B: P, t2 C: ¬P, t3 A: P. Is there an epistemic state consistent with the log where at t3 B knows P while C does not know that B knows P? Outline a forward-backward model-checking approach with Kripke structures and public announcements?
