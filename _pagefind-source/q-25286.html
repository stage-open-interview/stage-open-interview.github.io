You are given a rooted tree with N nodes, root = 1. Each node i has weight w[i]. You must select exactly K root-to-node paths that start at the root and terminate at some node in the tree. The paths must be vertex-disjoint except at the root. Each path's value is the sum of weights along that path. Return the maximum total value achievable and the endpoints of the K paths. Provide a DP formulation on the tree, specify dp[v][t] and how to combine children's contributions; include reconstruction?
