Given an array a1..an (n up to 2e5) and an integer k, partition into exactly k non-empty contiguous segments. The cost of a segment [l..r] is (sum(l..r))^2. Design an O(k n log C) DP using a Li Chao segment tree to minimize total cost. Explain the initialization, recurrence dp[k][j] = min_i dp[k-1][i] + (S[j]-S[i])^2, how lines L_i(x) = (dp[k-1][i]+S[i]^2) + (-2 S[i]) x are managed, and discuss potential pitfalls in practice?
