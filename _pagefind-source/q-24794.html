You process a stream of integers arriving in batches of size B. The window is fixed at size N. After each batch you append the B values and drop the oldest B values to keep N. After each batch determine if there exists a pair in the window with difference exactly K. Propose a data structure with (i) O(N) initialization, (ii) amortized O(B) updates, (iii) O(1) query, (iv) near-linear space, (v) two optimizations, (vi) a comparison to re-scanning. Provide asymptotics in N, B, K?
