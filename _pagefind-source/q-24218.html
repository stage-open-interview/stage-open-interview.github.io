Return the maximum total value and the list of chosen tasks. Propose an DP solution running in O(n log n + nK) time by sorting tasks by end time, computing p[i] as the last non-overlapping task before i via binary search, and using dp[i][t] = max(dp[i-1][t], dp[p[i]][t-1] + v_i). Include base cases and reconstruction?
