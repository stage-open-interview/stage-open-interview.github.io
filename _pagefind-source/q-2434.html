You're given an array A of length n and an integer k. Choose exactly k non-empty, non-overlapping subarrays. The score of a subarray [l..r] is sum(A[l..r]) - min(A[l..r]). Return the maximum total score and the list of subarray endpoints (l, r). Propose a DP formulation: dp[i][t] = max_{p in [t-1..i-1]} dp[p][t-1] + sum(A[p+1..i]) - min(A[p+1..i]). Base: dp[i][1] = sum(A[1..i]) - min(A[1..i]). Include reconstruction, base cases, and time/space complexity?
