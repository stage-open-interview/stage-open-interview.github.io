Wrap a C API that manages a configuration as an opaque handle. The C API provides: typedef struct Config Config; Config* config_new(const char* json); const char* config_to_json(const Config* cfg); void config_free(Config* cfg); The string returned by config_to_json is allocated with malloc and must be freed with free. Build a CGO wrapper in Go exposing: type ConfigHandle struct { ptr *C.Config } func CreateConfig(json string) (*ConfigHandle, error) func ConfigToJSON(h *ConfigHandle) (string, error) func (h *ConfigHandle) Close() error Requirements: - Allocate and free memory correctly, translate C strings to Go strings, and free buffers. - Ensure safe usage across two goroutines by calling CreateConfig in two goroutines, then pass to ConfigToJSON in parallel, and free the handles?
