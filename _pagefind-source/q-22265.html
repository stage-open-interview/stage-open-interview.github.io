Given a string s of length n and an array delCost[1..n] representing the deletion cost of each character, delete exactly k characters so that the remaining string is a palindrome. Minimize total deletion cost. Return the minimum cost and the resulting palindrome. Propose a DP dp[i][j][d] representing the minimal cost for s[i..j] after deleting exactly d characters, with transitions and reconstruction?
