If you step on a boost cell you may spend one boost to teleport to any unblocked cell in the same row or column. Count paths modulo 1e9+7. Propose a bottom-up DP with O(r c B) time using row/col prefix sums for boosted sources, and a memoized variant. Discuss edge cases?
