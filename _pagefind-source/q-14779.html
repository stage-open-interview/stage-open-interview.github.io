You're given a circular array A[0..n-1] and an integer k (1 <= k <= n). Partition the circle into exactly k non-empty, consecutive arcs that cover all elements. Each arc [s..e] contributes max(A[s..e]) - c, where c is a fixed per-arc cost. Return the maximum total value and the endpoints of the k arcs in circular order. Propose a DP approach that handles the circular wrap (e.g., fix a break, linearize, compute dp over A[i..i+n-1]), give recurrences and reconstruction, and discuss time/space?
