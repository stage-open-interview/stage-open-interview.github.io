You are given a rooted tree with N nodes. Each node v has weight w[v]. You must select exactly K node-disjoint simple paths located anywhere in the tree. The union of chosen paths must form a forest in which every connected component is a simple path (no branching). The score is the sum of weights of all nodes included in the paths. Return the maximum total weight and the list of paths as sequences of node IDs (endpoints to endpoints). Propose a bottom-up tree DP that computes dp[u][k][t], where k is the number of complete paths in the subtree of u and t ∈ {0,1,2} encodes whether u is unused (0), an endpoint that is closed inside the subtree (1), or an endpoint that can be extended to the parent (2). Include base cases, transitions over children, and reconstruction?
