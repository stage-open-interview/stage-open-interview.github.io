In a directed weighted graph with non-negative weights, some vertices are landmarks. Find the shortest s->t path that visits at least one landmark. Propose an O((N+M) log N) solution and implement a two-layer graph: state 0 (pre-landmark) and state 1 (landmark visited). Transitions follow edges; if a destination vertex is a landmark, you move to state 1. Return the shortest distance and the number of distinct shortest paths under this constraint, modulo 1e9+7, plus one exemplar path. Handle corner cases (no landmark reachable, s or t is a landmark, multiple shortest paths)?
