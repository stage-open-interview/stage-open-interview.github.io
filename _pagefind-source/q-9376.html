Given an array A[0..n-1], integer k, and minimum block length L, choose exactly k non-overlapping contiguous blocks, each of length at least L, in order. The value of a block is the sum of A within that block. Return the maximum total value and the endpoints of the blocks. Propose a DP with dp[i][t] representing the best total value for first i elements partitioned into t blocks, define the transition over block length s>=L, include reconstruction, and discuss time/space?
