Scenario: an append-only event stream arrives in batches of B events, each event labeled with a type from a large domain U. Design a data structure that, after each batch, returns the current top-k most frequent types with exact counts for those top-k. The structure must support O(B) batch processing, O(1) amortized per-event updates, near O(k log U) per-batch top-k retrieval, and O(U) space. Explain the combined use of a Space-Saving sketch to prune candidates and a per-batch exact consolidation to compute exact counts for the current top-k. Provide asymptotics in terms of n total events, U, B, and k, and discuss cache behavior as B varies?
