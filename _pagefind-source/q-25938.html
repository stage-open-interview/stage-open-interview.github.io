You're given a DAG with N nodes and M directed edges; each node v has value w[v]. You must select exactly k vertex-disjoint directed paths that start at a source (in-degree 0) and end at a sink (out-degree 0), maximizing the sum of weights of all visited nodes across all paths. Propose a DP in topological order using f[v][t] as the max total weight for t paths that end at v, plus how to reconstruct the paths?
