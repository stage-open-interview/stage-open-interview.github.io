Design a CGOA wrapper for an asynchronous C task API with cancellable handles and streaming progress callbacks. The C API provides: typedef struct TaskHandle TaskHandle; TaskHandle* start_task(const char* spec, void (*cb)(const char* json, size_t len, void* ctx), void* ctx); int cancel_task(TaskHandle*); The callback returns malloc-allocated JSON chunks. Provide a header, a minimal C implementation, a Go binding exposing StartTask(spec string) (chan string, chan error, func Stop()) that safely copies and frees data, and a Go program that runs three tasks concurrently, streams progress, and cancels one gracefully?
