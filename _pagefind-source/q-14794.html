Scenario: a manufacturing pipeline is a DAG of tasks. Each node i has duration d[i], each edge (u,v) has a setup s[u,v]. A task can start when all prerequisites finish and the maximum EFT[u]+s[u,v] over its incoming edges is achieved. EFT[v] = ES[v]+d[v], ES[v] = max_u(EFT[u]+s[u,v]); sources have EFT = d[i]. The project finish time is max EFT. Compute EFT for all nodes and the number of distinct critical schedules (mod 1e9+7), i.e., number of longest paths achieving the project finish time. Provide an O(N+M) DP?
