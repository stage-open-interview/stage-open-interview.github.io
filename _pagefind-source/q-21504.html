Given a DAG G=(V,E) representing a microservice call graph. Each edge (u,v) has a non-negative weight w(u,v). A message starts at source s. In one model, informed nodes can immediately forward along all outgoing edges, but v is only informed after sum of weights along a path from s to v. Compute earliest inform time for every node, i.e., the shortest path distance from s to each node. Provide an O(V+E) solution using a topological order and DP. Explain how you build the topological order and perform relaxations?
