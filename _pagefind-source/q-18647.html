Design a runtime-pluggable API gateway policy engine that enforces authentication, authorization, throttling, and response transformation. Policies are implemented as plugins loaded from a registry at runtime, allowing new policies to be added without redeploying the gateway. The engine must execute policies in a defined order, support short-circuiting, and report metrics per policy. Provide minimal interfaces, a registry, and a TypeScript usage example that wires together JwtAuthPolicy, RoleBasedPolicy, QuotaPolicy, and JsonTransformPolicy?
