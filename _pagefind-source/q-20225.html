Design and implement a tiny OS memory manager for a toy system with 8 physical frames and per-process 4-entry page tables. On a fault, allocate a free frame if available; otherwise evict the oldest frame (FIFO), writing back if dirty, and update both the victim and requester page tables. Provide a minimal C-like fault handler and an eviction function, plus a small test scenario where two processes alternate page touches to force eviction. How would you verify correctness and prevent thrashing?
