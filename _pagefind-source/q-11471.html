Design a runtime-pluggable query execution backend system for a data engine. Backends include CPU, GPU, and FPGA. Each backend must implement a minimal interface: execute(plan, ctx): ExecutionResult and supports(plan): boolean. A BackendRegistry must allow registering new backends by name at runtime and instantiating them via factories. The planner should pick a backend based on a policy (e.g., prefer GPU if available and plan is vectorizable). Provide interfaces, a registry, and a TypeScript usage example?
