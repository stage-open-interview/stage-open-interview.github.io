Design a runtime-extendable CLI command dispatcher that can load new subcommands as plugins without restarting the process. Each plugin implements a Command interface with execute(args: string[]): Promise<void> and help(): string. The dispatcher must support registering commands by name at runtime, isolating failures so one plugin cannot crash others, and injecting shared services. Provide minimal interfaces, a registry, and a TypeScript usage example. Discuss Plugin vs Strategy trade-offs and concurrency safety during plugin loading?
