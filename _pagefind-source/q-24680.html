Design a CGOA wrapper for a C streaming API: void start_streaming(const uint8_t* data, size_t len, void (*chunk_cb)(const char* json, void* ctx), void* ctx); The C library emits a JSON chunk for each processed record via chunk_cb; each chunk is allocated with malloc and must be freed by the receiver after copying. Provide: 1) a header and minimal C impl that registers a callback, streams chunks, and frees each JSON after callback; 2) a Go binding exposing StreamProcess(data []byte) error that wires the callback and streams input; and 3) a Go program that feeds a continuous data stream to a worker pool, collecting per chunk JSON into Go structures, freeing C memory, and propagating errors?
