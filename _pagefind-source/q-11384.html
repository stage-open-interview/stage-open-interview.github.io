Given a string s of length n and an integer k, partition s into exactly k non-empty substrings in order. Each substring’s cost is the minimum number of character changes needed to make it a palindrome. Return the minimal total cost and the cut positions. Propose a DP: precompute costPal[i][j] for all i<=j as s[i..j] palindrome changes, then dp[x][t] = min_{i<x} dp[i][t-1] + costPal[i][x-1], with base dp[0][0]=0; reconstruction pointers; discuss time/space?
