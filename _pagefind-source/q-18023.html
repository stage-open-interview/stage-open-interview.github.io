Incremental single-source shortest paths with batched edge insertions: You have a directed graph G=(V,E) with nonnegative weights. Edges are added only in batches of size B. After each batch you must output SSSP distances from a fixed source s to all nodes within a (1+ε) approximation. Propose a practical incremental structure that (i) preprocesses in O(m log n), (ii) amortized updates per batch of O(B log n / ε), (iii) answers dist(s,v) in O(1) for all v using stored buckets, (iv) uses near-linear space, (v) includes two optimizations, (vi) compares to re-running Dijkstra after every batch. Give asymptotics in n,m,B,ε?
