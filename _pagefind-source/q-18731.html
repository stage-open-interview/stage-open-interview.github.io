Design a CGOA wrapper for a C function that computes per-column statistics for a large matrix. The C API is int process_matrix(const double* data, size_t rows, size_t cols, char** out_json); data is row-major contiguous. The function returns JSON: [{\"column\":0,\"mean\":...,\"stddev\":...},...], malloc-allocated in C. Provide a header, a minimal C implementation using malloc for out_json, a Go binding exposing ComputeMatrixStats(matrix [][]float64) (string, error), and a Go program that streams matrices to a worker pool, ensuring memory ownership and error reporting?
