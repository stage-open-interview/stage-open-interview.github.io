Given a digit string s of length n and an integer k, partition s into exactly k non-empty substrings by placing k-1 cuts. Each substring is a decimal number with no leading zeros (except '0'). The partition's value is the sum of the numbers. Return the maximum sum and the cut positions. Propose a DP: define dp[i][t] = max sum for first i chars partitioned into t blocks; value(j,i) is the integer of s[j..i-1]. Provide recurrences, base cases, reconstruction, and discuss time/space?
