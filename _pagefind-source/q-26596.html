Across multiple hosts encoded in path (e.g., /logs/host1/.../app.log), write a robust one-liner (no external deps beyond standard UNIX tools) that, for the last 4 hours, outputs per host the number of unique ERROR events whose message matches /panic|segfault|oom|illegal/ case-insensitive. An event is unique by (host, first-line timestamp, message snippet up to 120 chars). Include gz archives and skip unreadable files. Output: <host>:<count> per line?
