Given a directed graph G=(V,E) with node costs d[v], start s, target t. First, detect cycles with DFS and report any cycle nodes; if no cycle, compute a topological order and then: for each v, earliestFinish[v] = max over u in preds(v) of earliestFinish[u] + d[v], with earliestFinish[s] = d[s]. Output earliestFinish[t]. Second, count the number of distinct s→t paths that achieve earliestFinish[t], modulo 1e9+7. Explain data structures, steps, and time complexity?
