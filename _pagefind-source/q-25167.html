Provide: 1) a header and minimal C glue implementing the bridge to Go callbacks; 2) a Go binding exposing StartStream() that returns a receive-only channel of blocks; 3) a Go program that uses a worker pool to compute per-block SHA-256 digests and a final batch checksum. Ensure memory ownership, error propagation, and clean shutdown?
