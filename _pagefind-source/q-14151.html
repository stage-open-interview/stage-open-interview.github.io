Provide a Go binding exposing: func SerializeInts(nums []int32) ([]byte, error) which allocates the needed buffer, calls the C function, and returns a Go slice containing the JSON. Manage memory ownership carefully (free any C-allocated memory) and demonstrate concurrency by invoking SerializeInts concurrently from two goroutines with different inputs. Include a minimal C header, C implementation, the Go binding, and a small Go program that runs in parallel?
