Given a required set T ⊆ V that must be true and a budget K, decide if there exists a model with T true and total cost ≤ K, and produce the minimum-cost model if possible. Explain a forward-chaining approach to minimize cost, handle cycles, and illustrate with a small 5-variable example showing two feasible models with different costs?
