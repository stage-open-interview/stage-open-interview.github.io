Find the minimum total distance of a path from S to T that visits S -> c1 -> c2 -> ... -> ck -> T in this exact order. If any leg is unreachable, return -1. Describe an O(N+M) approach that uses a topological sort and DP to compute the shortest path for each leg, and explain how you'd reconstruct the full path?
