You're given an array A of length n and an integer k. Partition A into exactly k non-empty contiguous subarrays. Each subarray's cost is the maximum element within that subarray. Return the minimum total cost and the partition endpoints. Propose a DP formulation with states dp[i][t] representing the minimum cost to partition the first i elements into t subarrays, with recurrence dp[i][t] = min_{p in [t-1..i-1]} dp[p][t-1] + max(A[p+1..i]). Base: dp[i][1] = max(A[1..i]). Describe reconstruction, base cases, and time/space complexity?
